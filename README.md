# AI步頻與動作數據分析 (Pose Estimation & Motion Analysis System)

這是一個專為短距離衝刺選手及田徑運動設計的進階運動分析平台。系統結合Pose Estimation與IMU技術，能夠自動從訓練影片中精準提取人體骨架節點，並同步整合 IMU 數據（如加速度、角速度）。透過視覺與感測器數據的雙重驗證，系統能提供步頻、步幅、爆發力等全方位的動作品質評估與視覺化，協助教練與選手優化起跑動作與衝刺機制。

## 核心功能

*   **人體姿勢預測**：利用深度學習模型自動追蹤影片中的關鍵節點（如膝蓋、髖關節、腳踝、肩膀、肘關節）。
*   **IMU 數據整合**：支援上傳並分析 IMU 數據，計算合成加速度、角速度變化及位移積分。
*   **動作數據圖表**：自動生成運動曲線圖表，方便觀察動作的規律性與爆發力。
*   **紀錄對比系統**：支援同時開啟兩筆紀錄進行骨架曲線或 IMU 數據的交叉比對，並可調整時間偏移（Offset）進行同步。
*   **選手紀錄管理**：系統化管理每位選手的歷史測試紀錄與分析影片。
*   **LINE通知**：分析完成後可透過 Line Notify 發送通知。

## 環境建置

### 1. 前置需求
*   **作業系統**：Windows 10/11 (建議)
*   **Python 版本**：Python 3.7
*   **資料庫**：SQL Server (需安裝 ODBC Driver 17 for SQL Server)
*   **硬體建議**：建議配備 NVIDIA GPU 以加速深度學習推論（需安裝對應之 CUDA/cuDNN）

### 2. 安裝步驟

1.  **複製專案**：
    ```bash
    git clone <repository_url>
    cd pose_estimation
    ```

2.  **安裝依賴套件**：
    ```bash
    pip install -r requirements.txt
    ```

3.  **資料庫設定**：
    *   請確保 SQL Server 已啟動。
    *   建立資料庫 `MotionAnalysis`。
    *   建立使用者 `MotionUser` 密碼為 `MotionUser` (或依據 `modules/db.py` 修改連線字串)。
    *   執行初始化腳本建立所需的資料表（Record, Player 等）。

## 使用操作

### 1. 啟動系統
執行以下指令啟動 Flask 網頁伺服器：
```bash
python app.py
```
啟動後，開啟瀏覽器造訪 `http://127.0.0.1:5000`。

### 2. 選手管理
*   在首頁可以新增選手資訊或選擇既有選手。
*   進入選手頁面後，點擊「新增紀錄」上傳分析影片與對應的 IMU CSV 檔案。

### 3. 執行分析
*   上傳成功後，系統會自動啟動分析流程：
    1.  **Pose Detection**：偵測骨架並計算關節角度。
    2.  **IMU Processing**：處理原始 IMU 數據並產出分析圖。
    3.  **Gait/Step Analysis**：計算步態指標。
*   分析完成後，可直接在網頁播放器查看標註骨架後的影片。

### 4. 數據對比
*   進入「交叉比對」頁面。
*   選擇兩位選手或同一選手的不同紀錄。
*   選擇比對類型（骨架角度或 IMU 數據）。
*   點擊「RUN COMPARE」產生對比圖表，並可使用同步功能對齊兩組數據的峰值。
